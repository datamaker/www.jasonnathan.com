### ENV variables
```
# in a bash script somewhere....
export CLIENT_ID="YOUR-WP.COM-APP-ID"
export CLIENT_SECRET="YOUR-WP.COM-APP-SECRET"
export WP_USERNAME="YOUR-WP.COM-USERNAME"
export WP_PASSWD="YOUR-WP.COM-PASSWORD"
```

```
meteor npm install
```
